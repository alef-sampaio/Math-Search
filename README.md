# Math-Seach
Jogo educativo de plataforma para ensinar números primos a crianças.
## Objetivo

Ajudar crianças a compreenderem o conceito de números primos por meio de desafios dentro de um jogo de plataforma.

## Tecnologias

- Godot
- GDScript
- Desenvolvimento de jogos 2D
- Lógica de programação

## Status
Completo e disponivel na versão mais recente
