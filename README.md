# Math-Search
Jogo educativo de plataforma para ensinar números primos a crianças.
## Objetivo

Ajudar crianças a compreenderem o conceito de números primos por meio de desafios dentro de um jogo de plataforma.

## Tecnologias

- Godot
- GDScript
- Desenvolvimento de jogos 2D
- Lógica de programação

## Funcionalidades

- Mecânica de plataforma 2D
- Ensino de números primos por meio de desafios
- Sistema de fases
- Interface voltada para crianças
- Versão jogável para Windows

## Controles

- Setas ou A/D: mover personagem OU
- Espaço: pular
- Tecla "i" para iniciar diálogo
- Tecla "o" para pular diálogo

## Status do projeto

Completo!

## Aprendizados

Durante o desenvolvimento do Math Search, foram aplicados conceitos de lógica de programação, organização de cenas no Godot, criação de mecânicas de jogo e desenvolvimento de uma proposta educativa baseada em gamificação.


## Como baixar e jogar

O **Math Search** está disponível para Windows por meio da aba de releases do repositório.

### Passo a passo

1. Acesse a página de releases do projeto:

[Baixar Math Search](https://github.com/alef-sampaio/Math-Seach/releases/tag/v1.0)

2. Na seção **Assets**, baixe o arquivo `.zip` do jogo.

> Atenção: não baixe os arquivos **Source code (zip)** ou **Source code (tar.gz)**, pois eles são apenas o código-fonte do projeto.

3. Depois de baixar o arquivo `.zip`, clique com o botão direito sobre ele e selecione:

```text
Extrair tudo
```

4. Abra a pasta extraída.

5. Execute o arquivo:

```text
Math.Search.exe
```

6. Pronto! O jogo será iniciado.

### Observações

- O jogo está disponível para **Windows**.
- Mantenha os arquivos `.exe` e `.pck` na mesma pasta.
- Caso o Windows mostre um aviso de segurança, clique em **Mais informações** e depois em **Executar assim mesmo**, caso tenha baixado o jogo pelo repositório oficial.

## Sobre o jogo

**Math Search** é um jogo educativo de plataforma desenvolvido para ensinar crianças sobre números primos de forma interativa e divertida.
